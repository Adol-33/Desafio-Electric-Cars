## Demo en línea

[Ver proyecto en GitHub Pages](https://adol-33.github.io/Desafio-Electric-Cars/)

## Descripción
Sitio web de autos electricos responsivo que utiliza CSS Grid y Flexbox para organizar la estructura principal y los productos.
Con Media Query para ser responsiva para dispositivos móviles, table y de escritorio. Incluye un menú, una barra de filtros, pie de pagina y paginación.

## Funcionalidades
- Diseño principal con CSS Grid.
- Autos organizados con Flexbox y tarjetas responsivas.
- Menú con Flexbox.
- Imágenes de autos adaptables al contenedor.
- Efectos hover en imagenes y botones.
- Adaptabilidad a diferentes tamaños de pantalla (media queries).
